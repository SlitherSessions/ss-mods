Slither.io Mods

Why another slither.io mod set? Simply put, I'm tired of other mods not working correctly on OSX and/or Safari so we decided to write our own.

These mods are written in-house by us. If there's a feature missing or something you'd like to see, send us a message at slither@kushview.net.

Features

  • IP Connect: connect to the same server as your friends.
  • Additional Skins
  • More coming soon...

Installing the Mods

Chrome Extension

Installation is cake. Just go to the Slither Sessions Official Extension page and click add to chrome.

Safari/Firefox

Installation on these browers uses Tampermonkey.

  1. Install Tampermonkey (Safari/ Firefox)
  2. If needed, double click the dowloaded file from the tampermonkey site
  3. Add the Slither Sessions User Script
  4. Click the update or install button
